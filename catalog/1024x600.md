**Included Meters (10):**

- 102G5_Kenwood Rev S+M
- 103G5_Kenwood Spectrum
- 104G5_Marschal Spectrum
- 105G5_Technics S+M
- 106G5_OldPipe Spectrum
- 107G5_475A Oscil S+M
- 108G5_PeppyP S+M
- 109G5_Nixie S+M
- 110G5_Old Spectrum S+M

**Download:** [1024x600_g5_110_ms.zip](../templates_peppy_spectrum/1024/600/1024x600_g5_110_ms.zip)

**Install (both required):**
1. Extract the zip file
2. Copy `templates/1024x600_g5_110_ms/` to `/data/INTERNAL/peppy_screensaver/templates/`
3. Copy `templates_spectrum/1024x600_g5_110_ms/` to `/data/INTERNAL/peppy_screensaver/templates_spectrum/`
