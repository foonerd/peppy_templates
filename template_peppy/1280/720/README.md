# 720 Templates

## 1280x720_custom_01

![1280x720_custom_01](previews/1280x720_custom_01.png)

|----------|-------|
| Meter Name | vertical-turntable-blue |
| Meter Type | linear |
| Extended Config | Yes |
| Spectrum | No |
| Album Art | Yes |

2. Extract each to the path shown next to its download link
3. Select in plugin settings
