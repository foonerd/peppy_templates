# 400 Templates

VU Meter templates for PeppyMeter Screensaver.

## 1280x400_black_blue_square

![1280x400_black_blue_square](previews/1280x400_black_blue_square.png)

| Property | Value |
|----------|-------|
| Meter Name | black-blue-square |
| Meter Type | circular |
| Extended Config | Yes |
| Spectrum | No |
| Album Art | Yes |

**Download:** [1280x400_black_blue_square.zip](1280x400_black_blue_square.zip)

**Install:** Extract and copy folder to `/data/INTERNAL/peppy_screensaver/templates/`
